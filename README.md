## Portfolio (Next.js)

제출용 포트폴리오 템플릿입니다.

### 로컬 실행 (3001)

```bash
cd portfolio
npm run dev
```

브라우저에서 `http://localhost:3001` 로 접속하세요.

### 제출용 정보 수정 위치

- **전체 텍스트/링크/프로젝트 데이터**: `src/lib/site.ts`
- **홈**: `src/app/page.tsx`
- **프로젝트 목록**: `src/app/projects/page.tsx`
- **연락처**: `src/app/contact/page.tsx`

### 빌드/프로덕션 실행 (3001)

```bash
npm run build
npm run start
```
